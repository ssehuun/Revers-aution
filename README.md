# **프로젝트 설명**

* 챗봇 기반 휴대폰 요금제 추천 시스템 (2017.03 ~ 2017.06)
* 페이스북 메세지 API를 활용한 휴대폰 요금제 추천 서비스 개발


### **<디렉토리 설명>**

**app.js** - NodeJs 서버를 구성

**public** - 템플릿 및 날짜 라이브러리

**views** - HTML ejs 템플릿 파일
